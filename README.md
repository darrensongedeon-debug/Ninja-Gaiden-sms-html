# Ninja-Gaiden-sms-html
Html port of the Sega Master System version of Ninja Gaiden. Click start on the top left to start. Controls: Arrow keys to move, Z key to attack and X key to jump. P key to pause or you can also press the 3 lines button on the top left to pause and also the audio button on the top left to turn the volume on or off.
https://darrensongedeon-debug.github.io/Ninja-Gaiden-sms-html/
